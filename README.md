# Tacview 场景编辑器

纯前端单文件 HTML，无需安装，双击打开即用。在浏览器中可视化编辑 Tacview 战术场景，支持导出标准 ACMI 文件或实时发送到 Tacview 软件。

左侧为实体类型库，中间为地图画布，右侧为实体列表与属性面板。点击地图放置实体，选中后编辑坐标、速度、阵营、航路点等属性，支持撤销/重做。

支持的对象类型包括空中平台（战斗机、预警机、直升机等）、地面平台（坦克、防空系统、建筑等）、海上平台（战舰、航母、潜艇等）以及导航标记。每类实体可配置航路点路径，支持循环巡逻与任务完成后消失两种模式。

雷达支持扇形搜索与球形防空区两种模式，可绑定到任意实体跟随移动。可以显示数据链效果，可配置消息类型、延迟、激活时段等参数，底部面板提供拓扑图与链路列表两种视图。

工具栏内置场景预览播放器，可按时间轴回放实体运动、雷达旋转与数据链激活状态，支持尾迹与外推预测显示。导出功能生成标准 ACMI 文件，发送功能通过本地 Python 脚本以 UDP 遥测流实时推送到 Tacview。
